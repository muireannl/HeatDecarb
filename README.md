# Risk-aware decarbonisation of residential heat

This repository contains the Python code used in the paper:

> Lynch, M. (2026). Risk-aware decarbonisation of residential heat: a case study of Ireland.

## Purpose
The code is provided for transparency and reproducibility. It reflects research code developed during the course of the analysis and has not been refactored for general use.

## Requirements
The analysis was run using Python 3.12. Required packages are listed in `requirements.txt`.

## Data
All data used in the analysis are publicly available from the sources cited in the paper and code.
